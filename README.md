
#### 命令别名

你可以使用以下命令名：

- `78dm`

#### 参数

`<搜索关键词> page` 是你想要查找的任何内容，可以包含空格。最后一个参数如是数字则识别为page参数，默认为一页

#### 示例

-   搜索特定的高达模型：
    ```
    78dm MGEX 强袭自由 3
    ```

## ⚙️ 配置

本插件无需任何配置，即装即用。

## 🧩 依赖

- `requests`
- `beautifulsoup4`

这些依赖项已在 `requirements.txt` 中声明，AstrBot 会自动为您安装。
