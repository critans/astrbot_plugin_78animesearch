
#### 命令别名

你可以使用以下任意一个命令名：

- `78dm`
- `78动漫`
- `模型搜索`

#### 参数

`<搜索关键词>` 是你想要查找的任何内容，可以包含空格。

#### 示例

-   搜索特定的高达模型：
    ```
    78dm MGEX 强袭自由
    ```
-   使用中文命令名：
    ```
    78动漫 RG 海牛
    ```
-   带上前缀搜索：
    ```
    /模型搜索 HG 风灵
    ```

## ⚙️ 配置

本插件无需任何配置，即装即用。

## 🧩 依赖

- `requests`
- `beautifulsoup4`

这些依赖项已在 `requirements.txt` 中声明，AstrBot 会自动为您安装。
